### 📝 Sobre o Projeto
O **Cordel Moderno** é uma experiência web imersiva dedicada à exposição da rica obra de **Milton Duarte**. Este projeto foi cuidadosamente desenvolvido para aplicar e demonstrar conhecimentos em desenvolvimento web front-end, unindo a tradição da literatura de cordel à sofisticação do design digital contemporâneo.

O site se destaca pela utilização de **HTML5 100% semântico**, garantindo não apenas a estrutura lógica e acessibilidade do conteúdo, mas também otimização para mecanismos de busca (SEO). Além disso, incorpora **efeitos de imagem parallax** através de **CSS3**, criando uma profundidade visual dinâmica que enriquece a jornada de leitura e a interação do usuário com o conteúdo.

### 🚀 Tecnologias Utilizadas
- ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=flat&logo=html5&logoColor=white) **HTML5** (Estrutura Semântica)
- ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=flat&logo=css3&logoColor=white) **CSS3** (Estilização, Responsividade e Efeitos Parallax)

### 🛠️ Pré-requisitos
Para visualizar ou trabalhar com este projeto localmente, você precisará de:
*   Uma IDE de sua preferência (ex: Visual Studio Code).
*   Um navegador web moderno (Chrome, Firefox, Edge, Safari).

### 📥 Instalação e Execução
1.  Clone o repositório para sua máquina local:
    ```bash
    git clone https://github.com/jeanbaialopes/jeanbaialopes.git
    
2.  Navegue até o diretório do projeto:
    ```bash
    cd jeanbaialopes
    ```
3.  Abra o arquivo `index.html` diretamente no seu navegador ou utilize uma extensão como "Live Server" (para VS Code) para visualização.

### 💻 Exemplo de Uso
O projeto é uma página estática interativa. Ao rolar a página, observe como as imagens de fundo se movem em velocidades diferentes do conteúdo textual, criando o efeito de parallax que adiciona profundidade e dinamismo à experiência de leitura.

### 📝 About the Project
**Cordel Moderno** is an immersive web experience dedicated to showcasing the rich work of **Milton Duarte**. This project was carefully developed to apply and demonstrate knowledge in front-end web development, blending the tradition of "cordel" literature with the sophistication of contemporary digital design.

The website stands out for its use of **100% semantic HTML5**, ensuring not only the logical structure and accessibility of the content but also optimization for search engines (SEO). Furthermore, it incorporates **parallax image effects** through **CSS3**, creating a dynamic visual depth that enriches the reading journey and user interaction with the content.

### 🚀 Technologies Used
- ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=flat&logo=html5&logoColor=white) **HTML5** (Semantic Structure)
- ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=flat&logo=css3&logoColor=white) **CSS3** (Styling, Responsiveness, and Parallax Effects)

### 🛠️ Prerequisites
To view or work with this project locally, you will need:
*   An IDE of your choice (e.g., Visual Studio Code).
*   A modern web browser (Chrome, Firefox, Edge, Safari).

### 📥 Installation and Execution
1.  Clone the repository to your local machine:
    ```bash
    git clone https://github.com/jeanbaialopes/jeanbaialopes.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd jeanbaialopes
    ```
3.  Open the `index.html` file directly in your browser or use an extension like "Live Server" (for VS Code) for viewing.

### 💻 Usage Example
The project is an interactive static page. As you scroll down the page, observe how the background images move at different speeds than the textual content, creating the parallax effect that adds depth and dynamism to the reading experience.
